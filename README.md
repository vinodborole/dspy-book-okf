# dspy-book-okf

An **[OKF](https://github.com/GoogleCloudPlatform/knowledge-catalog/blob/main/okf/SPEC.md) knowledge bundle** of [https://dspy.ai/](https://dspy.ai/) — ready for agents to read and for you to chat with.

```bash
pip install okf-kit
okf get dspy-book
okf chat dspy-book --provider ollama
```

Built and kept fresh with [okf-kit](https://github.com/vinodborole/okf-kit).
